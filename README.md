🎬 Flask REST API – Video Management System

A simple RESTful API built with Flask, Flask-RESTful, and SQLAlchemy that allows users to create, read, update, and delete video records.

This project demonstrates how to build a structured backend API with database integration and proper HTTP status handling.

🚀 Features
Create a video
Get a video by ID
Update a video
Delete a video
SQLite database integration
Input validation with reqparse
Proper HTTP status codes
Environment variable support using .env
Cross-Origin Resource Sharing (CORS) enabled
🛠️ Technologies Used
Python
Flask
Flask-RESTful
Flask-SQLAlchemy
SQLite
Werkzeug (Password Hashing)
python-dotenv
Flask-CORS
🌐 CORS Support

This API includes CORS (Cross-Origin Resource Sharing) support using the Flask-CORS
 library. This allows your frontend applications, running on different domains or ports, to make requests to the API without being blocked by the browser’s same-origin policy.
